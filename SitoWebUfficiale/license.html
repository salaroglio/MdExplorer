Cosa significa questa licenza?

La licenza MIT è una delle licenze open source più permissive. Significa che:

  • Puoi usare mdExplorer per qualsiasi scopo, commerciale o non commerciale
  • Puoi modificare il codice sorgente secondo le tue necessità
  • Puoi distribuire il software originale o modificato
  • Puoi includere mdExplorer in software proprietario
  • Devi includere l'avviso di copyright e la licenza in tutte le copie
  • Non hai garanzie - il software è fornito "così com'è"
  • Gli autori non sono responsabili per eventuali danni o problemi
