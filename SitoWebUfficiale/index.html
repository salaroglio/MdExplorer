Gestione Professionale di Documenti Markdown

mdExplorer è una suite completa per la gestione, editing e conversione di documenti Markdown con supporto per PlantUML, Git integration e export multipli.

Download Gratuito Codice Sorgente

Funzionalità Principali

Editor Markdown Avanzato

Editor WYSIWYG con supporto per sintassi estesa, preview live e highlighting del codice.

Gestione Progetti

Organizza documenti in progetti con struttura ad albero, tag e ricerca avanzata.

PlantUML Integration

Genera diagrammi UML direttamente dai tuoi documenti Markdown.

Export Multipli

Esporta in PDF, Word, HTML con template personalizzabili.

Git Integration

Versioning integrato per tracciare modifiche e collaborare.

AI Assistant

Assistente AI integrato per generazione contenuti e TOC automatici.
